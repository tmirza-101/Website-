# NAMC Education Portal

## North American Mobeds Council Educational Website

This is a comprehensive educational website for the North American Mobeds Council (NAMC), designed as a university-style learning platform for Zoroastrian religious education.

## Features

### Dual Portal System
- **Teachers Portal**: Complete course management, student tracking, grading, and content creation
- **Students Portal**: Course enrollment, assignment submission, progress tracking, and resource access

### Main Features

#### For Teachers:
- Dashboard with real-time statistics
- Course creation and management
- Student enrollment and progress tracking
- Assignment and grade management
- Material upload and organization
- Calendar and scheduling
- Messaging system
- Reports and analytics

#### For Students:
- Personalized learning dashboard
- Course enrollment and access
- Assignment viewing and submission
- Grade tracking
- Progress monitoring
- Resource library access
- Calendar integration
- Direct messaging with instructors

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with CSS Grid and Flexbox
- **Typography**: Inter font family from Google Fonts
- **Icons**: Feather Icons (SVG)
- **Design**: Modern, responsive, mobile-friendly

## Getting Started

1. **Add the NAMC Logo** (Optional but recommended):
   - Download the logo from https://www.namcmobeds.org
   - Save it as `namc-logo.png` in the Website folder
   - See `LOGO_INSTRUCTIONS.md` for detailed steps

2. Open `index.html` in a web browser
3. Choose either Teachers Portal or Students Portal
4. Use the login forms to access the respective dashboards

### Demo Access
Currently, the authentication is set up for demonstration purposes. Any email and password will allow access to the dashboards.
